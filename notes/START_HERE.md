- ✅ 先看项目文档，建立整体认识

- ✅ 逐步修改，每次改一个东西
- ✅ 用小参数快速迭代
- ✅ 频繁检查 Wandb 的结果

### 最佳实践

```
1. 每次修改前 git commit（备份）
2. 运行 quick_test.sh 验证基线
3. 修改一个参数 / 代码行
4. 运行完整训练看结果
5. 在 Wandb 对比
6. 记录发现
7. 重复 2-6
```
